# Vue3 Storybook Example

Example for a basic Storybook integration in a Vue3 application.

## Project setup

### Step 1. Install dependencies

```
pnpm run install
```

### Step 2. Compilation

#### Compiles and hot-reloads for development

```
pnpm run serve
```

#### Compiles and minifies for production

```
pnpm run build
```
